### Development and running the API Server
---

1. clone
2. move to ``` cd schools-api ```
3. copy ``` development.env ``` to ``` .env ```
4. run ``` yarn && docker-compose up --build ```

### Production

3. copy ``` prod.env ``` to ``` .env ```
